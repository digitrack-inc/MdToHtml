C++23 / HDX AGV

Autonomous Forklift Control System

HDX AGV

포크리프트 무인 운반 차량(AGV) C++ 제어 시스템. 실시간 30ms 제어 루프와 Zenoh 브로커리스 통신 기반의 산업용 자율 주행 소프트웨어.

30ms 실시간 루프 Zenoh P2P 통신 C++23 Fail-Safe 설계

주요 기능

⚡

30ms 실시간 제어

절대 시간 스케줄링으로 drift를 방지하는 독립 제어 루프. 제어 주기가 통신 지연의 영향을 받지 않습니다.

🔗

Zenoh 브로커리스 통신

ROS2 스타일 Node/Publisher/Subscriber 구조. 브로커 없는 P2P Pub/Sub 통신으로 <1ms 지연.

🛡

Fail-Safe 설계

장애 발생 시 즉시 제로 명령 출력. std::expected 기반 타입 안전 에러 처리.

🌳

BehaviorTree 임무 계획

BehaviorTree.CPP 연동으로 복잡한 자율 주행 임무를 구조적으로 구성합니다. (선택적 빌드)

📊

텔레메트리 녹화

SQLite3 로컬 저장 및 InfluxDB 연동(선택)으로 운행 데이터를 실시간 기록합니다.

⚙️

계층형 아키텍처

Service → Communication → Control → Hardware 4계층 구조로 각 레이어가 명확히 분리됩니다.

  • Visual Studio 2022 — C++ 데스크톱 워크로드, MSVC v143 컴파일러

버전 확인:

conan --version   # 2.0 이상
cmake --version   # 3.20 이상

설치 및 빌드

1

저장소 클론

git clone <repository-url>
cd hdx_agv
2

Conan 프로파일 설정

MSVC v143(Visual Studio 2022) 컴파일러를 사용하는 기본 프로파일을 생성합니다.

$profileDir = Join-Path $HOME ".conan2/profiles"
New-Item -ItemType Directory -Force -Path $profileDir | Out-Null

@"
[settings]
compiler=msvc
compiler.version=193
compiler.update=8
compiler.runtime=dynamic
compiler.cppstd=23
arch=x86_64
build_type=Release
os=Windows

[conf]
sqlite3/*:tools.build:cflags+=["/utf-8"]
sqlite3/*:tools.build:cxxflags+=["/utf-8"]
"@ | Set-Content (Join-Path $profileDir "default")
3

의존성 설치 및 빌드

# Conan 의존성 설치 (첫 실행 시 2~3분 소요)
conan install . -of build --build=missing -s compiler.cppstd=23

# CMake 구성
cmake --preset conan-release

# 빌드 (1~2분 소요)
cmake --build build --config Release

아키텍처

시스템은 4개 계층으로 구성됩니다. 실시간 제어 루프와 Pub/Sub 모니터링 경로를 분리하여 제어 주기가 통신 지연의 영향을 받지 않습니다.

┌─────────────────────────────────────────────────┐
│              Service Layer                      │
│  BehaviorTree | Navigation | Planning           │
├─────────────────────────────────────────────────┤
│           Communication Layer                   │
│      Zenoh P2P Transport (브로커리스)            │
├─────────────────────────────────────────────────┤
│              Control Layer                      │
│  HdxForkliftControllerService (30ms 독립 스레드) │
│  DriveController | ForkController               │
├─────────────────────────────────────────────────┤
│             Hardware Layer                      │
│  ZapiClient(CAN) | WireClient(UDP) | DigitalIO  │
└─────────────────────────────────────────────────┘

문제 해결

ErrorConan 의존성 누락 오류

ERROR: Missing prebuilt package
conan install . -of build --build=missing -s compiler.cppstd=23

ErrorCMake 컴파일러 미발견 오류

CMake Error: Could not find CMAKE_C_COMPILER

Visual Studio 2022 C++ 데스크톱 워크로드 및 MSVC v143 컴파일러가 설치되어 있는지 확인합니다.

ErrorDLL 미발견 런타임 오류

The code execution cannot proceed because *.dll was not found

실행 파일과 동일한 디렉토리(build/services/*/Release/)에서 실행합니다. zenohc.dll은 빌드 시 자동으로 복사됩니다.
